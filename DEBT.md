## How to use

- **Adding debt:** append a row. Include the file path. If you're leaving
  it knowingly in a PR, link the PR in the row.
- **Paying it down:** flip status to `in-progress`, then `done` with the
  commit. Don't silently delete — a reviewer should see it was retired.
- **Re-audit cadence:** D9 and D12 are re-checked on every hermes-agent
  version bump (run `POST /api/reprobe`, diff `CapabilityFlags`, re-read
  `docs/CAPABILITY_COVERAGE.md`).
